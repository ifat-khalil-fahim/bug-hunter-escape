# 🐞 Bug Hunter Escape

A story-driven coding escape game built with **HTML, CSS, and JavaScript**.

As I recently learned JavaScript,, I created this project to practice DOM manipulation, game logic, timers, local storage, and user interaction while building something fun and creative.

## 🎮 Live Demo

🔗 https://bughunterescape.netlify.app

---

## 📖 About The Game

Instead of creating a traditional quiz game, I added a story-driven experience set in a fictional cyber world called Sector-7. The game combines coding challenges, voice narration, timers, scoring, and progression mechanics to make the experience more enjoyable and engaging for players.

Each correct answer unlocks another door and brings you closer to freedom. Every mistake pushes you closer to being permanently trapped inside Sector-7.

Can you survive the system and escape before time runs out?
